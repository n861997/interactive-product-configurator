## Project Overview

**Author:** Nicole Mottier

**Repository Access:** Private — available to recruiters upon request

Developed an interactive web-based tool that allows customers to customize the visual appearance of the Averionics Typis 2.0 3D printer by selecting colors for individual components. The system dynamically updates a visual preview and submits the selected configuration via email for order processing.

This tool enables mass customization without requiring manual configuration by staff, improving user experience and operational efficiency.

## Core Functionality

- Interactive selection of individual printer components

- Real-time visual preview of chosen colors

- Support for multi-part customization

- Submission of configuration data to the company via email

- Designed for ease of use by non-technical customers

## Technical Implementation

**Frontend Development**

- JavaScript / jQuery

- Interactive image mapping

- Event-driven UI updates

- DOM manipulation for dynamic rendering

**Libraries & Tools**

- jQuery

- Image mapping via Imagemapster

- HTML/CSS for layout and styling

**Data Handling & Integration**

- Captures user selections programmatically

- Formats configuration data for transmission

- Sends results via email for fulfillment workflow

## Technical Skills Demonstrated

- Frontend web development

- Interactive UI/UX design

- State management for user selections

- Client-side data collection

- Integration with business processes

- Testing and deployment of customer-facing features

## Project Goals

- Provide customers with a clear visualization of customized products

- Reduce order errors caused by manual specification

- Streamline internal processing of custom builds

- Enhance perceived product personalization

## Testing & Validation

The tool was tested for:

- Accuracy of component selection mapping

- Visual consistency across browsers

- Usability for first-time users

- Correct transmission of configuration details

- Edge cases such as incomplete selections
